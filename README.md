# Orbital

**Orbital** 是一个专为 Linux 移动设备设计的现代化、轻量级桌面环境（Dashboard）。

它基于 **Qt 6** 和 **QML** 构建，旨在以极低的资源占用提供流畅的动画效果和直观的交互体验。Orbital 能够让用户实时掌握设备运行状态，并轻松管理无线网络与硬件设置。

## 主要功能

Orbital 目前包含以下核心模块：

### 系统资源监控

* **CPU**: 实时显示 CPU 占用率与负载情况。
* **内存**: 监控 RAM 使用量及剩余空间。
* **存储**: 硬盘/文件系统空间使用概览。

### 电源与硬件

* **电池监控**: 实时读取电池电量百分比及充放电状态。
* **亮度调节**: 支持通过滑动条直接调节屏幕背光亮度（支持 PWM 或系统文件接口）。

### 网络管理

* **流量监控**: 实时显示网络上传与下载速率。
* **WiFi 管理**:
* 自动扫描周围可用的 WiFi 热点。
* 支持连接到指定网络（支持 WPA/WPA2 加密）。
* 直观的信号强度指示与连接状态反馈。

## 构建与运行

### 前置依赖

确保你的开发环境或目标设备已安装以下依赖：

* Qt 6 (Base, Declarative)
* C++ 编译器 (GCC/Clang)
* NetworkManager (用于 WiFi 功能支持)

### 编译步骤

```bash
# 1. 克隆项目
git clone https://github.com/your-username/orbital.git
cd orbital

# 2. 创建构建目录
mkdir build && cd build

# 3. 配置与编译
cmake ..
make -j$(nproc)

# 4. 运行
./appOrbital

# 可选：注册为服务

```
